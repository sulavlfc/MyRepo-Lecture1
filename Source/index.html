I've listed out some steps i followed for this assignment
  1. Create a repo
  2. Clone the repo
  3. Make some changes in local
  4. Add the changes for commit
  5. Commit the changes
  6. Push the changes to remote

The images for above steps are shown below

Here are some of the git commands

  • git clone : Clones the remote repo to the local system
  • git add : Stages the changes
  • git commit : Commits the changes made in the local repo
  • git push origin master : Pushes the commited changes to master branch of remote
